# HostLoca - XAMPP Controller
HostLoca is an open source Python desktop application designed to make working with XAMPP faster, safer, and more developer friendly.  XAMPP is a great local development environment, but it often comes with frustrations: losing your htdocs folder during reinstall, struggling with database imports, and juggling root passwords. HostLoca solves these problems by automating backups, simplifying database management, and letting you set up custom superusers that match your production environment.  With HostLoca, you can:  Keep your projects safe in a custom HTDOCS folder outside XAMPP.  Automatically import .sql and .sql.gz files into MySQL without manual phpMyAdmin steps.  Configure a unique superuser alongside default accounts for quick switching between local and production credentials.  It’s built by a developer, for developers - and now it's open source so the community can help it grow.
